## Better Voice 1.0.2

### Fixes

- **In-app updates actually work now.** The Sparkle updater was bundled and configured but never activated, so earlier builds silently never checked for updates. Better Voice now checks on its own and adds **Check for Updates…** (and an **Update to X…** row when one is waiting) to the menu bar — release notes plus one-click Install & Relaunch.
- Carries forward 1.0.1's reliable first-run permission grants and the dictation hotkey fixes.

> One-time note: because 1.0 and 1.0.1 shipped without a working updater, this update can't install itself on those versions — grab 1.0.2 once from [voice.baselinemakes.com](https://voice.baselinemakes.com). From 1.0.2 onward, updates install in place.

### Requirements

macOS 26+, Apple silicon. Apple Intelligence recommended for the zero-setup default.
